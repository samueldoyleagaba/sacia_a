# SACIA_A

**Student Automation & Coding Intelligent Assistant**

SACIA_A is a free, open-source Windows automation tool designed to help students
learn automation by performing real tasks such as file organization and batch renaming.

## SDG Alignment
- SDG 4: Quality Education
- SDG 9: Industry, Innovation & Infrastructure

## Features
- File organization automation
- Batch renaming
- Automation logic explanation

## Run
```bash
python sacia.py
